Methodology
1. Electrochemical equations implemented from published alkaline electrolyzer studies (LeRoy 1980, Zeng & Zhang 2010).
2. Voltage losses computed using Butler-Volmer activation model and KOH ohmic resistance model.
3. Hydrogen production rate obtained via Faraday's law with bubble coverage and Faradaic-efficiency corrections.
4. Parametric studies performed over temperature (20–90°C), pressure (1–10 bar) and current density (0.05–1.0 A/cm²).

Levelized Cost of Hydrogen (LCOH)
The cost cards above only add up the per-kg CAPEX share you set on a slider. LCOH does the full lifecycle math instead — it spreads the actual plant CAPEX over its lifetime using a capital recovery factor at a given discount rate, divides by how much H₂ the plant actually makes at a realistic capacity factor, and adds that to the running electricity/water/O&M cost. This is the number normally quoted when comparing hydrogen production routes.

TOPSIS Best Operating Point
A Pareto front, by definition, has no single winner — every point on it beats every other point on something. TOPSIS breaks that tie by measuring each candidate's distance to an ideal solution (best value of every objective, even if no real point achieves all of them at once) and to an anti-ideal one, then picks whichever point is closest to the good corner and farthest from the bad one. Set the weights to whatever actually matters for this plant and re-rank.

Local Explanation — Why This Prediction?
Breaks the current prediction down into a contribution from each input (T, P, j) using exact Shapley values — the marginal effect of each variable averaged over all 6 possible orders in which you could "turn it on" from a neutral reference point. The three contributions always add up exactly to the gap between the reference prediction and the current one, so nothing here is approximate.
